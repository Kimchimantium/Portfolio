What can he do?

저는 Python 개발의 모든 면에 깊이 파고드는 것을 목표로 삼고, Udemy Angela Yu의 파이썬 개발자 양성 프로그램을 성공적으로 마친 개발자입니다. 이 강의를 통해 Python의 기초부터 심화 문법, 데이터사이언스, 웹 개발, GUI 프로그래밍, 데이터 크롤링, 업무 자동화에 이르기까지 다양한 분야에서 Python을 활용할 수 있는 실력을 갖추게 되었습니다.

제 학습 여정은 단순히 이론을 넘어서, 100개의 실제 프로젝트를 직접 구현하면서 Python을 이용한 개발에 필요한 실질적인 경험을 쌓는 데 중점을 두었습니다. 이 과정을 통해 저는 블랙잭 게임, 뱀 게임, 퐁 게임 같은 재미있는 게임부터 틴더 자동 스와이핑 봇, 링크드인 자동 구직 지원 앱, 생일 축하 이메일 자동 전송 시스템 등 실생활에서 유용하게 쓰일 수 있는 다양한 애플리케이션 개발에 이르기까지 폭넓은 프로젝트를 완성했습니다.

데이터사이언스 분야에서는 Numpy, Pandas, Matplotlib, Seaborn, Plotly, Scikit-Learn 등을 활용하여 데이터 분석과 시각화 프로젝트를 수행했습니다. 웹 개발에 있어서는 HTML/CSS의 기초부터 시작해 Bootstrap, Flask, Jinja templating engine, SQLite, SQLAlchemy, PostgreSQL, REST API를 통한 백엔드 개발에 이르기까지 전반적인 웹 개발 스킬을 갖추게 되었습니다.

GUI 프로그래밍 분야에서는 Turtle과 Tkinter를 활용한 다양한 데스크탑 애플리케이션 개발 경험도 쌓았으며, Python을 이용한 데이터 크롤링 기술로 BeautifulSoup과 Selenium을 사용하여 웹 데이터를 추출하는 방법도 마스터했습니다.

이 모든 학습 과정을 통해 저는 Python 개발자로서의 깊이 있는 전문 지식과 다양한 프로젝트 경험을 바탕으로 어떠한 개발 문제에도 유연하게 대응할 수 있는 능력을 갖추게 되었습니다. Python 개발에 대한 열정과 지속적인 학습을 통해 앞으로도 더 많은 분야에서 혁신적인 솔루션을 제공하고자 합니다.
